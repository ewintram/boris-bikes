User Story:

As a person,
So that I can use a bike,
I'd like a docking station to release a bike.
As a person,
So that I can use a good bike,
I'd like to see if a bike is working

Functional Representation:

Object              Messages
Person(user)
Bike                is_working?
Docking Station     has_spare_bike?
                    release_bike
                    return_bike
